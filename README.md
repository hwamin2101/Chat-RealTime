# Real-Time AI Chat App

A real-time chat application with streaming AI responses.

## Features
- Real-time messaging (Socket.IO)
- AI replies streamed token-by-token 
- One-on-one & group chats
- Reply to messages, typing indicators, online status
- Image/file upload (Cloudinary)
- Dark/Light mode + fully responsive
- Secure auth (JWT + HttpOnly cookies)

## Tech Stack
- Frontend: React 19, TypeScript, Tailwind, Shadcn/UI, Zustand
- Backend: Node.js, Express, Socket.IO, MongoDB
- AI: Streaming responses (Gemini)
- Deploy: Vercel
